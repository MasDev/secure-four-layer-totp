--------- To open the system for testing
 
✅ 1) Commands → Use CMD / Terminal

These steps are executed in Command Prompt (Windows) or Terminal (Mac/Linux).

✔ Step-by-step in CMD:
1. Clone repository
git clone https://github.com/MasDev/secure-four-layer-totp

2. Go inside project
cd secure-four-layer-totp

3. Install dependencies
npm install

4. Start the server
node server/server.js

✅ 2) Open Browser

After starting the server, open your browser and go to:

http://localhost:3000/login.html
